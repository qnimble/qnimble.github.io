Quick Start Guide

  1. Install the latest Arduino IDE (if not already installed)

  2. Open Arduino

  3. Open the preferences dialog by clicking on File and then Preferences:

    In the "Additional Boards Manager URLS" field, put:

    https://qnimble.com/package_qnimble_index.json

    If you already have url's there, add a comma (,) at the end of the list, then add the line.

  4. Open the Boards Manager by clicking on Tools, Board and then Boards Manager

  5. Type "qnimble" into the text box at the top

  6. Then "qNimble iMXRT Boards" should show up, and click "install"

  7. Once the installation is complete, qNimble iMXRT Boards should be listed as INSTALLED and you can click on Close.

  8. If you do not have an empty sketch, click on File and New to open blank sketch.

  9. Click on Tools, Board, qNimble and select Quarto

  10. Put your code in your sketch and when ready click on Upload to load the code onto the Quart If you want simple program to blink the front panel LED, paste in the following code:

asdfasd.js
void setup() {
// put your setup code here, to run once:
}
void loop() {
static unsigned long lastrun;
if (millis() > lastrun + 500) { //Run once every 500ms
toggleLEDGreen(); //toggle green LED;
lastrun = lastrun + 500;
}
}
