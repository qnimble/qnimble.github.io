Triggers

The following functions are available for the two Trigger lines. The "X" in the function name must be replaced by the trigger number (1 or 2):

  • setTriggerXDirection
  • setTriggerXLow
  • setTriggerXHigh
  • getTriggerX
  • enableInterruptTriggerX
  • disableInterruptTriggerX

setTriggerXDirection#

void setTrigger2Direction(io_direction_t direction);

This function sets the trigger to be either an input or an output. By default, all triggers are inputs.

  • direction Can either be PIN_DIRECTION_OUTPUT or PIN_DIRECTION_INPUT

Example#

setTrigger1Direction(PIN_DIRECTION_INPUT);
setTrigger2Direction(PIN_DIRECTION_OUTPUT);

setTriggerXLow#

void setTriggerXLow(void);

This function sets the trigger output to low or 0V. If the trigger is not configured as an output, this function has no effect.

Example#

setTrigger1Low();
setTrigger2Low();

setTriggerXHigh#

void setTriggerXHigh(void);

This function sets the trigger output to high or 3.3V. If the trigger is not configured as an output, this function has no effect.

Example#

setTrigger1High();
setTrigger2High();

getTriggerX#

bool getTriggerX(void);

The function returns true if the trigger is high and false if it is low. This function works when the trigger is configured as an input or an output.

Example#

if (getTrigger1()) {
//Trigger1 is high
...
} else {
//Trigger1 is low
}
bool result = getTrigger2(); //store Trigger2 level.

enableInterruptTriggerX#

void enableInterruptTriggerX(bool rising_edge, void (*cb_function)(void) );

This function configures the trigger to call a function on the rising (or falling) edge of the trigger line.

  • rising_edge If set to true, function is called on a rising edge of the signal. If false, then function called on falling edge
  • cb_function Callback function to run on the rising (or falling) edge of the trigger

Example#

void setup() {
enableInterruptTrigger1(true, gotTrigger);
}
void gotTrigger(void) {}
...
}

disableInterruptTriggerX#

void disableInterruptTriggerX(void);

This function disables the trigger's interrupt function.

Example#

void watchTrigger(bool on) {
if (on == true) {
enableInterruptTrigger1(true, gotTrigger);
//call gotTrigger when Trigger 1 goes high
} else {
disableADC1(); //turn off calling gotTrigger
}
}
void gotTrigger(void) {
...
}
