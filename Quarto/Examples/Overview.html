Software Examples

Below are links to detailed write-ups of software examples for using the Quarto:

  • PID Servo: 100kHz Proprtional-Integrator-Differential (PID) Servo, with integrator hold
  • Pulsed Output Waveform: Send a pulsed arbitrary waveform on a trigger event

More examples of using the Quarto in various ways are available inside the Arduino IDE. Select File, Examples and go to the "Examples for Quarto" section, as shown below. From there you can open, edit, compile and upload any of the examples.

Alternatively, you can find these examples at the quarto-examples repository on GitHub.

Do you have an idea for an example you don't see? Please let us know by either posting on our forum or contacting us.
